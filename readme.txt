Add more responsive support on tablet for ELEMENTOR

== Description ==

This plugin do only one thing :

It gives breakpoint control for tablet like elementor already do it for smartphone.


== Installation ==

1. Install Elementor plugin.
2. Install More Responsive Tablet 4 Elementor (this plugin)
3. Create a page with elementor, add column then go to advanced tab and responsive area
4. Define your column width on tablet
